Kebijakan Privasi

Terakhir diperbarui: 2026

Maintenance Reminder ("kami") menghargai privasi Anda. Aplikasi ini dirancang untuk membantu pengguna mengelola jadwal perawatan tanpa mengumpulkan atau menyimpan data pribadi.

1. Pengumpulan Data

Kami tidak mengumpulkan, menyimpan, atau membagikan data pribadi pengguna. Aplikasi berjalan sepenuhnya di perangkat Anda dan tidak mengirim data ke server eksternal.

2. Data Pengguna

Semua data seperti jadwal perawatan, pengingat, dan catatan disimpan secara lokal di perangkat Anda. Kami tidak memiliki akses terhadap data tersebut.

3. Perizinan

Aplikasi ini dapat meminta izin berikut:

  • Izin Notifikasi: Digunakan untuk mengirim pengingat jadwal perawatan.

Izin tersebut digunakan hanya untuk fungsi utama aplikasi dan tidak digunakan untuk pelacakan atau pengumpulan data.

4. Layanan Pihak Ketiga

Aplikasi ini tidak menggunakan layanan pihak ketiga untuk analitik, pelacakan, maupun iklan.

5. Keamanan Data

Karena semua data disimpan secara lokal, keamanan data bergantung pada keamanan perangkat Anda. Kami tidak menyimpan atau mengirim data ke server mana pun.

6. Privasi Anak

Aplikasi ini tidak secara sengaja mengumpulkan informasi dari anak di bawah usia 13 tahun.

7. Perubahan Kebijakan

Kebijakan ini dapat diperbarui sewaktu-waktu. Perubahan akan ditampilkan pada halaman ini.
